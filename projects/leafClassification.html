Project: Leaf Classification

Business problem to solve

This project is about classifying tree leaves.

Machine learning problem

The underlying problem is a multiclass problem. Each leaf will be classified into ONE class only, one out of 99 classes to be exact.

Evaluation metric used: LogLoss

Data Source

Taken from the official Kaggle website.
